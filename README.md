# Admission Prediction

## Overview
This project predicts university admission chances based on academic and profile features using machine learning techniques.

## Dataset
The dataset contains student academic information such as test scores, GPA, and other relevant attributes.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Methodology
- Data preprocessing and cleaning
- Exploratory data analysis
- Model training and evaluation

## Results
The model performance was evaluated using standard metrics to assess prediction accuracy.
